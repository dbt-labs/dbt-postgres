## dbt-postgres 1.9.0-a1 - August 01, 2024

### Features

* Add tests for cross-database `cast` macro
* Cross-database `date` macro
* Add support for Python 3.12

### Fixes

* Fix the semicolon semantics for indexes while respecting other bug fix
* Default to psycopg2-binary and allow overriding to psycopg2 via DBT_PSYCOPG2_NAME (restores previous behavior)
* Fix `persist_docs` for `materialized_view` materializations. Previously, using this configuration with materialized view models would lead to an error.

### Under the Hood

* Add support for experimental record/replay testing.
